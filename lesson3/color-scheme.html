Color Scheme:

#0B7AB8 [primary]

This blue color would be used for call to action[CTA] buttons, backgound color for navigation buttons on hover

#45494E [primary]

This gray color compliments teh blue perfectly and makes up the logo. It would be used for backgounds that aren't "whitesmoke" colored and the navigations and footer background

whitesmoke [secondary]

This color would be used as the main background color for the body

#FFB30F [accent]

The subtle touch of the orange balances the blue and grey. It would be used for highlighting texts such as the active nav icon.
